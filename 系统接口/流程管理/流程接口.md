# 流程管理模块接口文档

## 1. 问题验证流程

**接口路径**: `/api/tack/issues/{id}/verify`

**请求方法**: `POST`

### 入参

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | int | 是 | 问题ID |
| environment | string | 是 | 验证环境 |
| result | string | 是 | 验证结果，pass/fail |
| description | string | 否 | 验证描述 |
| attachment_ids | array | 否 | 附件ID列表 |

### 出参

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | int | 状态码，0表示成功 |
| message | string | 返回消息 |
| data | object | 验证结果信息 |
| data.id | int | 验证记录ID |
| data.issue_id | int | 问题ID |
| data.environment | string | 验证环境 |
| data.result | string | 验证结果 |
| data.status | string | 问题新状态 |
| data.user_id | int | 验证人ID |
| data.created_at | string | 验证时间 |

## 8. 获取问题轨迹

**接口路径**: `/api/tack/issues/{id}/timeline`

**请求方法**: `GET`

### 入参

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | int | 是 | 问题ID |

### 出参

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | int | 状态码，0表示成功 |
| message | string | 返回消息 |
| data | object | 轨迹信息 |
| data.issue_id | int | 问题ID |
| data.current_status | string | 当前状态 |
| data.environments | array | 环境验证记录 |
| data.environments[].name | string | 环境名称 |
| data.environments[].status | string | 验证状态 |
| data.environments[].verifier | object | 验证人信息 |
| data.environments[].verified_at | string | 验证时间 |
| data.environments[].result | string | 验证结果 |
| data.environments[].description | string | 验证描述 |
| data.environments[].attachments | array | 附件列表 |
| data.history | array | 完整历史记录 |
| data.rollback_records | array | 回退记录 |
