# 问题管理模块接口文档

## 1. 创建问题

**接口路径**: `/api/tack/issues`

**请求方法**: `POST`

### 入参

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| project_id | int | 是 | 项目ID |
| requirement_id | int | 否 | 需求ID |
| title | string | 是 | 问题标题 |
| description | string | 否 | 问题描述 |
| type | string | 是 | 问题类型 |
| severity | string | 是 | 严重程度 |
| environment | string | 是 | 发现环境 |
| version | string | 否 | 发现版本 |
| assignee_id | int | 否 | 负责人ID |
| attachment_ids | array | 否 | 附件ID列表 |

### 出参

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | int | 状态码，0表示成功 |
| message | string | 返回消息 |
| data | object | 问题信息 |
| data.id | int | 问题ID |
| data.project_id | int | 项目ID |
| data.requirement_id | int | 需求ID |
| data.title | string | 问题标题 |
| data.description | string | 问题描述 |
| data.type | string | 问题类型 |
| data.severity | string | 严重程度 |
| data.status | string | 问题状态 |
| data.environment | string | 发现环境 |
| data.version | string | 发现版本 |
| data.assignee_id | int | 负责人ID |
| data.creator_id | int | 创建人ID |
| data.created_at | string | 创建时间 |
| data.updated_at | string | 更新时间 |

---

## 2. 获取问题列表

**接口路径**: `/api/tack/issues`

**请求方法**: `GET`

### 入参

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| project_id | int | 否 | 项目ID筛选 |
| requirement_id | int | 否 | 需求ID筛选 |
| status | string | 否 | 状态筛选 |
| assignee_id | int | 否 | 负责人ID筛选 |
| severity | string | 否 | 严重程度筛选 |
| keyword | string | 否 | 关键词搜索 |
| page | int | 否 | 页码，默认1 |
| page_size | int | 否 | 每页数量，默认10 |
| sort_field | string | 否 | 排序字段 |
| sort_order | string | 否 | 排序方向，asc/desc |

### 出参

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | int | 状态码，0表示成功 |
| message | string | 返回消息 |
| data | object | 问题列表信息 |
| data.list | array | 问题列表 |
| data.list[].id | int | 问题ID |
| data.list[].title | string | 问题标题 |
| data.list[].type | string | 问题类型 |
| data.list[].severity | string | 严重程度 |
| data.list[].status | string | 问题状态 |
| data.list[].assignee_id | int | 负责人ID |
| data.list[].assignee_name | string | 负责人名称 |
| data.list[].project_id | int | 项目ID |
| data.list[].project_name | string | 项目名称 |
| data.list[].created_at | string | 创建时间 |
| data.list[].updated_at | string | 更新时间 |
| data.total | int | 总数量 |
