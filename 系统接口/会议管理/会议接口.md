# 会议管理模块接口文档

## 1. 创建会议纪要

**接口路径**: `/api/tack/meetings`

**请求方法**: `POST`

### 入参

| 参数名                 | 类型   | 必填 | 说明                                  |
| ---------------------- | ------ | ---- | ------------------------------------- |
| title                  | string | 是   | 会议主题                              |
| project_id             | int    | 是   | 所属项目 ID                           |
| meeting_date           | string | 是   | 会议日期，格式 YYYY-MM-DD             |
| participants           | array  | 是   | 参与人 ID 列表                        |
| status                 | string | 否   | 状态，pending/discussed，默认 pending |
| content                | string | 是   | 会议内容                              |
| conclusion             | string | 否   | 会议结论                              |
| meeting_link           | string | 否   | 腾讯会议链接                          |
| attachment             | string | 否   | 附件名称                              |
| related_requirement_id | int    | 否   | 关联需求 ID                           |

### 出参

| 参数名                      | 类型   | 说明               |
| --------------------------- | ------ | ------------------ |
| code                        | int    | 状态码，0 表示成功 |
| message                     | string | 返回消息           |
| data                        | object | 会议信息           |
| data.id                     | int    | 会议 ID            |
| data.title                  | string | 会议主题           |
| data.project_id             | int    | 项目 ID            |
| data.meeting_date           | string | 会议日期           |
| data.participants           | array  | 参与人列表         |
| data.status                 | string | 状态               |
| data.content                | string | 会议内容           |
| data.conclusion             | string | 会议结论           |
| data.meeting_link           | string | 腾讯会议链接       |
| data.attachment             | string | 附件               |
| data.related_requirement_id | int    | 关联需求 ID        |
| data.created_at             | string | 创建时间           |

---

## 2. 获取会议纪要列表

**接口路径**: `/api/tack/meetings`

**请求方法**: `GET`

### 入参

| 参数名     | 类型   | 必填 | 说明              |
| ---------- | ------ | ---- | ----------------- |
| page       | int    | 否   | 页码，默认 1      |
| page_size  | int    | 否   | 每页数量，默认 10 |
| project_id | int    | 否   | 项目 ID 筛选      |
| status     | string | 否   | 状态筛选          |
| keyword    | string | 否   | 关键词搜索        |

### 出参

| 参数名                   | 类型   | 说明               |
| ------------------------ | ------ | ------------------ |
| code                     | int    | 状态码，0 表示成功 |
| message                  | string | 返回消息           |
| data                     | object | 会议列表信息       |
| data.list                | array  | 会议列表           |
| data.list[].id           | int    | 会议 ID            |
| data.list[].title        | string | 会议主题           |
| data.list[].project_id   | int    | 项目 ID            |
| data.list[].project      | string | 项目名称           |
| data.list[].meeting_date | string | 会议日期           |
| data.list[].participants | array  | 参与人列表         |
| data.list[].status       | string | 状态               |
| data.list[].content      | string | 会议内容           |
| data.list[].conclusion   | string | 会议结论           |
| data.list[].meeting_link | string | 腾讯会议链接       |
| data.list[].attachment   | string | 附件               |
| data.list[].created_at   | string | 创建时间           |
| data.total               | int    | 总数量             |

---

## 6. 上传会议附件

**接口路径**: `/api/tack/meetings/{id}/attachment`

**请求方法**: `POST`

### 入参

| 参数名 | 类型 | 必填 | 说明     |
| ------ | ---- | ---- | -------- |
| id     | int  | 是   | 会议 ID  |
| file   | file | 是   | 附件文件 |

### 出参

| 参数名        | 类型   | 说明               |
| ------------- | ------ | ------------------ |
| code          | int    | 状态码，0 表示成功 |
| message       | string | 返回消息           |
| data          | object | 上传结果           |
| data.filename | string | 文件名             |
| data.url      | string | 文件访问 URL       |
