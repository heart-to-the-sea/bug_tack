# 个人待办管理模块接口文档

## 1. 获取待办列表

**接口路径**: `/api/todos`

**请求方法**: `GET`

### 入参

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| status | string | 否 | 状态筛选，pending/completed |
| priority | string | 否 | 优先级筛选，high/medium/low |
| project_id | int | 否 | 项目ID筛选 |
| deadline_from | string | 否 | 截止时间开始 |
| deadline_to | string | 否 | 截止时间结束 |
| sort | string | 否 | 排序字段，deadline/priority/created_at |
| order | string | 否 | 排序方向，asc/desc |
| page | int | 否 | 页码，默认1 |
| page_size | int | 否 | 每页数量，默认10 |

### 出参

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | int | 状态码，0表示成功 |
| message | string | 返回消息 |
| data | object | 待办列表信息 |
| data.list | array | 待办列表 |
| data.list[].id | int | 待办ID |
| data.list[].issue_id | int | 关联问题ID |
| data.list[].issue | object | 问题信息 |
| data.list[].issue.title | string | 问题标题 |
| data.list[].issue.status | string | 问题状态 |
| data.list[].issue.severity | string | 严重程度 |
| data.list[].priority | string | 优先级 |
| data.list[].deadline | string | 截止时间 |
| data.list[].status | string | 待办状态 |
| data.list[].remark | string | 备注 |
| data.list[].created_at | string | 创建时间 |
| data.total | int | 总数量 |

---
