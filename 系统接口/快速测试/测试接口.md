# 快速测试管理模块接口文档

## 1. 提交测试项

**接口路径**: `/api/tack/testing/submit`

**请求方法**: `POST`

### 入参

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| issue_id | int | 是 | 问题ID |
| environment | string | 是 | 测试环境 |
| description | string | 否 | 测试描述 |

### 出参

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | int | 状态码，0表示成功 |
| message | string | 返回消息 |
| data | object | 测试任务信息 |
| data.id | int | 测试任务ID |
| data.issue_id | int | 问题ID |
| data.environment | string | 测试环境 |
| data.status | string | 状态 |
| data.queue_order | int | 队列顺序 |
| data.created_at | string | 创建时间 |

## 3. 获取测试队列

**接口路径**: `/api/tack/testing/queue`

**请求方法**: `GET`

### 入参

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| environment | string | 是 | 测试环境 |
| status | string | 否 | 状态筛选，pending/in_progress/completed |

### 出参

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | int | 状态码，0表示成功 |
| message | string | 返回消息 |
| data | object | 队列信息 |
| data.environment | string | 测试环境 |
| data.total | int | 总数量 |
| data.pending | int | 待测试数量 |
| data.in_progress | int | 测试中数量 |
| data.completed | int | 已完成数量 |
| data.tasks | array | 测试任务列表 |
| data.tasks[].id | int | 任务ID |
| data.tasks[].issue_id | int | 问题ID |
| data.tasks[].issue_title | string | 问题标题 |
| data.tasks[].status | string | 状态 |
| data.tasks[].queue_order | int | 队列顺序 |
| data.tasks[].created_at | string | 创建时间 |

## 5. 执行测试

**接口路径**: `/api/tack/testing/{id}/execute`

**请求方法**: `POST`

### 入参

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | int | 是 | 测试任务ID |
| result | string | 是 | 测试结果，pass/fail |
| description | string | 否 | 测试描述 |
| attachment_ids | array | 否 | 附件ID列表 |
| auto_next | boolean | 否 | 是否自动跳转下一项，默认true |

### 出参

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | int | 状态码，0表示成功 |
| message | string | 返回消息 |
| data | object | 测试结果 |
| data.id | int | 测试记录ID |
| data.task_id | int | 任务ID |
| data.result | string | 测试结果 |
| data.next_task_id | int | 下一任务ID |
| data.is_last | boolean | 是否最后一个 |
| data.queue_progress | object | 队列进度 |
