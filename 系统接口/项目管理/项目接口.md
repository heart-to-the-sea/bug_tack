# 项目管理模块接口文档

## 1. 创建项目

**接口路径**: `/api/projects`

**请求方法**: `POST`

### 入参

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| name | string | 是 | 项目名称 |
| description | string | 否 | 项目描述 |
| leader_ids | array | 是 | 项目负责人ID列表 |
| member_ids | array | 否 | 项目成员ID列表 |

### 出参

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | int | 状态码，0表示成功 |
| message | string | 返回消息 |
| data | object | 项目信息 |
| data.id | int | 项目ID |
| data.name | string | 项目名称 |
| data.description | string | 项目描述 |
| data.leader_ids | array | 项目负责人ID列表 |
| data.member_ids | array | 项目成员ID列表 |
| data.created_at | string | 创建时间 |
| data.updated_at | string | 更新时间 |

---

## 2. 获取项目列表

**接口路径**: `/api/projects`

**请求方法**: `GET`

### 入参

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| page | int | 否 | 页码，默认1 |
| page_size | int | 否 | 每页数量，默认10 |
| keyword | string | 否 | 关键词搜索 |

### 出参

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | int | 状态码，0表示成功 |
| message | string | 返回消息 |
| data | object | 项目列表信息 |
| data.list | array | 项目列表 |
| data.list[].id | int | 项目ID |
| data.list[].name | string | 项目名称 |
| data.list[].description | string | 项目描述 |
| data.list[].leader_ids | array | 项目负责人ID列表 |
| data.list[].member_count | int | 成员数量 |
| data.list[].issue_count | int | 问题数量 |
| data.list[].created_at | string | 创建时间 |
| data.total | int | 总数量 |

---

## 6. 添加项目成员

**接口路径**: `/api/projects/{id}/members`

**请求方法**: `POST`

### 入参

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | int | 是 | 项目ID |
| user_ids | array | 是 | 要添加的用户ID列表 |
| role | string | 否 | 角色，默认member |

### 出参

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | int | 状态码，0表示成功 |
| message | string | 返回消息 |
| data | object | 结果信息 |
| data.added_count | int | 添加成功的数量 |

## 8. 创建需求

**接口路径**: `/api/projects/{project_id}/requirements`

**请求方法**: `POST`

### 入参

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| project_id | int | 是 | 项目ID |
| title | string | 是 | 需求标题 |
| description | string | 否 | 需求描述 |
| priority | string | 否 | 优先级，P0/P1/P2 |

### 出参

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | int | 状态码，0表示成功 |
| message | string | 返回消息 |
| data | object | 需求信息 |
| data.id | int | 需求ID |
| data.project_id | int | 项目ID |
| data.title | string | 需求标题 |
| data.description | string | 需求描述 |
| data.priority | string | 优先级 |
| data.status | string | 状态 |
| data.created_at | string | 创建时间 |
