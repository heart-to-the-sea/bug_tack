# 项目管理模块接口文档

## 1. 创建项目

**接口路径**: `/api/tack/projects`

**请求方法**: `POST`

### 入参

| 参数名      | 类型   | 必填 | 说明               |
| ----------- | ------ | ---- | ------------------ |
| name        | string | 是   | 项目名称           |
| description | string | 否   | 项目描述           |
| leader_ids  | array  | 是   | 项目负责人 ID 列表 |
| member_ids  | array  | 否   | 项目成员 ID 列表   |

### 出参

| 参数名           | 类型   | 说明               |
| ---------------- | ------ | ------------------ |
| code             | int    | 状态码，0 表示成功 |
| message          | string | 返回消息           |
| data             | object | 项目信息           |
| data.id          | int    | 项目 ID            |
| data.name        | string | 项目名称           |
| data.description | string | 项目描述           |
| data.leader_ids  | array  | 项目负责人 ID 列表 |
| data.member_ids  | array  | 项目成员 ID 列表   |
| data.created_at  | string | 创建时间           |
| data.updated_at  | string | 更新时间           |

---

## 2. 获取项目列表

**接口路径**: `/api/tack/projects`

**请求方法**: `GET`

### 入参

| 参数名    | 类型   | 必填 | 说明              |
| --------- | ------ | ---- | ----------------- |
| page      | int    | 否   | 页码，默认 1      |
| page_size | int    | 否   | 每页数量，默认 10 |
| keyword   | string | 否   | 关键词搜索        |

### 出参

| 参数名                   | 类型   | 说明               |
| ------------------------ | ------ | ------------------ |
| code                     | int    | 状态码，0 表示成功 |
| message                  | string | 返回消息           |
| data                     | object | 项目列表信息       |
| data.list                | array  | 项目列表           |
| data.list[].id           | int    | 项目 ID            |
| data.list[].name         | string | 项目名称           |
| data.list[].description  | string | 项目描述           |
| data.list[].leader_ids   | array  | 项目负责人 ID 列表 |
| data.list[].member_count | int    | 成员数量           |
| data.list[].issue_count  | int    | 问题数量           |
| data.list[].created_at   | string | 创建时间           |
| data.total               | int    | 总数量             |
