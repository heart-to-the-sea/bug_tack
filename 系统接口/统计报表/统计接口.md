# 统计报表模块接口文档

## 1. 问题状态统计

**接口路径**: `/api/statistics/issues/by-status`

**请求方法**: `GET`

### 入参

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| project_id | int | 否 | 项目ID筛选 |
| start_date | string | 否 | 开始日期 |
| end_date | string | 否 | 结束日期 |

### 出参

| 参数名 | 类型 | 说明 |
|--------|------|------|
| code | int | 状态码，0表示成功 |
| message | string | 返回消息 |
| data | object | 统计数据 |
| data.total | int | 问题总数 |
| data.by_status | array | 按状态统计 |
| data.by_status[].status | string | 状态名称 |
| data.by_status[].count | int | 数量 |
| data.by_status[].percentage | float | 占比 |

---
